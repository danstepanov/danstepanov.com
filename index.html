Personal Projects

I always make sure I make time for personal projects.

Project Reach

This is a secure & discrete communication platform I made for reporters who work in dangerous areas. I built Reach because I believe that every person is entitled to freedom of speech. Project Reach is both a web app and a native Android app built using React and and React-Native. Reach is currently under development but even so, it encrypts data using SSH and protects users by disguising itself as a different app as well as tying accounts to devices rather than people. Testing is currenly being conducted between a professor at Harvard and users in Syria.

Anycdote

This platform was inspired by Humans of New York. Anycdote allows users to pin personal stories to geographic locations. Other users can then experience stories that close to them by proximity. Anycdote is written completely in Swift and relies on a Parse backend.
